Panduan

Memulai Aplikasi

1. Login Pertama Kali

Buka aplikasi dan masukkan NIP serta kata sandi yang telah diberikan oleh admin sekolah.

2. Dashboard Overview

Dashboard menampilkan jadwal mengajar hari ini, jam kerja, dan status presensi terkini.

3. Navigasi Aplikasi

Gunakan menu bawah untuk berpindah antar halaman: Home, Riwayat, Presensi, Izin, dan Profil.

Panduan Presensi

1. Lokasi Valid

Pastikan Anda berada dalam radius sekolah (maksimal 100 meter dari gedung sekolah).

2. Wajah Jelas

Posisikan wajah Anda dengan jelas di kamera. Pastikan pencahayaan cukup dan tidak ada bayangan.

3. Jam Presensi

Presensi masuk dapat dilakukan 30 menit sebelum jam mengajar. Presensi keluar setelah selesai mengajar.

4. Notifikasi

Anda akan menerima notifikasi jika presensi berhasil atau ada masalah yang perlu diperbaiki.

Panduan Izin

1. Jenis Izin

Pilih jenis izin: Sakit (dengan surat dokter), Cuti Pribadi, atau Dinas Luar.

2. Pengisian Form

Isi tanggal mulai dan selesai, serta berikan keterangan yang jelas dan lengkap.

3. Lampiran Dokumen

Upload dokumen pendukung (surat dokter, surat tugas, dll) dalam format JPG atau PDF maksimal 5MB.

4. Status Persetujuan

Monitor status pengajuan di tab Riwayat. Status: Menunggu, Disetujui, atau Ditolak.

Troubleshooting

GPS Tidak Terdeteksi

Pastikan GPS di perangkat Anda aktif dan berikan izin lokasi untuk aplikasi.

Kamera Tidak Berfungsi

Periksa izin kamera di pengaturan perangkat dan restart aplikasi.

Lupa Kata Sandi

Hubungi admin sekolah untuk reset kata sandi. Tidak ada fitur lupa sandi otomatis.

Data Tidak Sync

Pastikan koneksi internet stabil dan coba refresh dengan menarik ke bawah pada dashboard.
